# Jadoo — Travel Agency Landing Page

A clean, responsive travel agency landing page built with plain HTML, CSS, and JavaScript. Inspired by the Jadoo Travel Website UI design from Figma Community.

## Sections

| Section | Description |
|---|---|
| **Navbar** | Sticky navigation with logo, links, login/signup, and mobile hamburger menu |
| **Hero** | Full headline with animated floating planes and traveler illustration |
| **Services** | 4 service cards — Weather, Flights, Events, Customization |
| **Top Destinations** | 3 destination cards with image, name, price, and trip duration |
| **Book in 3 Steps** | Step-by-step booking guide with a trip UI card preview |
| **Testimonials** | Expandable testimonial cards with dot navigation |
| **Partner Brands** | Airline/travel brand logo strip |
| **Subscribe** | Email subscription form with gradient card |
| **Footer** | Links, social icons, and app store badges |

---

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Flexbox, CSS Grid, custom properties, keyframe animations
- **JavaScript** — Vanilla JS, no frameworks or libraries
- **Fonts** — [Volkhov](https://fonts.google.com/specimen/Volkhov) (headings), [Poppins](https://fonts.google.com/specimen/Poppins) (body)
- **Icons** — [Font Awesome 6](https://fontawesome.com/)

---

## Features

- Fully responsive — works on desktop, tablet, and mobile
- Smooth scroll navigation
- Mobile slide-in menu
- Animated floating plane illustrations in hero
- Like button toggle on trip card
- Subscribe form with validation feedback
- Sticky header with scroll shadow effect

---

## How to Run

No build tools or dependencies needed. Just open the file in your browser:

```
Right-click index.html → Open with → Browser
```

Or use VS Code's **Live Server** extension for hot reload during development.
